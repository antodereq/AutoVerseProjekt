# AutoVerse Backend

Prosty backend dla rekomendatora AutoVerse.

## Instalacja

1. Otwórz terminal w katalogu `backend`
2. Uruchom:
   ```bash
   npm install
   ```

## Start

```bash
npm start
```

Backend będzie dostępny na `http://localhost:8080`.

### Endpoint

- `GET /api/cars` - zwraca dane samochodów do rekomendacji
